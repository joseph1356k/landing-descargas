# Carpeta de descargas

Aquí vive el binario de la app (actualmente `you-0.37.apk`).

El botón de descarga apunta a este archivo servido desde GitHub
(ver `DOWNLOAD_URL` en `js/config.js`). Al subir una versión nueva,
agrega el archivo aquí y actualiza esa constante (y los `href` de
respaldo en `index.html`).
